Thumbnail

이미지 그리드는 여러 개의 썸네일 이미지로 구성됩니다.
이미지에 포함된 추가 정보와 링크는 사용자가 작업을 수행하는 데 도움이 됩니다.

anatomy

Options

Time line

영상 재생 시간을 표시한 것으로 영상 썸네일 왼쪽 하단에 위치합니다. 카테고리에 따라 DIM 박스를 넣고 빼는 게 가능합니다.

Icon Area

오른쪽 하단에 위치하게 되는 이미지 형태 아이콘입니다. 아래 순서대로 포토, 화보, 영상, 그래픽 그리고 전체 보기 아이콘으로 구성되어 있습니다.

Text Area

이미지 위에 텍스트 사용하는 것은 권장되지 않습니다. 다만 필요한 경우에 따라 적용 할 수 있습니다.

  • Font color : #FFFFFF
  • Font weight : Medium More than
Dimmer

딤에는 Full과 Gradient 두 가지가 있습니다. Full 딤처리 적용은 영상에 주로 사용하고 있습니다. 하단 고정되는 Gradient는 텍스트 또는 하단에 주요 요소가 배치 되는 경우에 사용합니다.

  • Full BG : #000000,40%
  • Gradient : #000000,0% - #000000,100%
Border

흰배경의 이미지와 배경을 구분하기 위해서 이미지에는 라인이 필수로 들어가야 합니다.

Line : #000000,4% (1px)

Behaviors

Video

영상 썸네일에 마우스 오버 했을 때 나타나는 영상 플레이 버튼입니다.

keyword

메인 톱뉴스에 노출되는 키워드로 썸네일 오른쪽 하단에 위치합니다. PC에서 마우스 오버 시 키워드 배경 컬러가 포인트 됩니다.

Time line + keyword

메인 톱뉴스에서 영상 썸네일일 경우에는 영상 재생시간과 키워드를 같이 노출할 수 있습니다.

Aspect Ratio

노출되는 뉴스 이미지 비율은 6:4(3:2)비율을 고정으로 사용 하고 있고, 영상 비율은 16:9를 사용하고 있습니다.
비율이 맞지 않는 이미지의 경우 아래 방식으로 노출 됩니다.

Image Exposure Method

가로형 이미지

가로형 이미지를 등록한다면 양 옆이 잘려 노출됩니다.
(4:3), (3:2), (16:9), (7:3), (3:1) 비율이 이에 속합니다.

  • Line : #000000,4% (1px)
    • 세로형 이미지

    세로형 이미지를 등록한다면 높이를 기준으로
    이미지 노출하고 배경 컬러를 넣어줍니다.
    (2:3), (3:4), (1:1) 비율이 이에 속합니다.

  • BG : #F9F9F9
  • Line : #000000,4% (1px)
    • 이미지 없는 경우

    이미지 없는 경우에는 로고 심볼만 노출합니다.
    인물 등 특수한 경우에 사용됩니다.

  • BG : #F5F5F5
  • Line : #000000,4% (1px)
    • 뉴스 속보 이미지

    긴급 속보 사진 없는 케이스에 사용됩니다.
